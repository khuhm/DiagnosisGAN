## Requirements
* Python >= 3.6
* Pytorch >= 1.2
* Ubuntu >= 16.04
* TITAN Xp

## Installation
```
git clone https://github.com/khuhm/DiagnosisGAN.git
cd DiagnosisGAN/
```

## Usage
```
python train_cls.py
python train_syn.py
```
